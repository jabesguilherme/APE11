Comandos e suas finalidades sem o uso do Flow

git init: Inicializa um novo repositório Git local.
git clone [URL]: Clona um repositório existente para o diretório local.
git add [arquivo(s)]: Adiciona arquivos ao stage, preparando-os para o commit.
git commit -m "mensagem": Registra as alterações no repositório local com uma mensagem descritiva.
git status: Exibe o estado atual do repositório, mostrando quais arquivos foram modificados, adicionados ou estão no stage.
git log: Mostra o histórico de commits.
git branch [nome]: Cria um novo ramo com o nome especificado.
git checkout [ramo]: Muda para o ramo especificado.
git merge [ramo]: Combina as alterações de um ramo com o ramo atual.
git pull origin [ramo]: Atualiza o repositório local com as alterações do repositório remoto.
git push origin [ramo]: Envia os commits locais para o repositório remoto no ramo especificado.
